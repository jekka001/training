Task
Написать свою, простую, эффективную реализацию алгоритма Hash Map с открытой адресацией, для ключей типа int и значений типа long. 
Реализовать операции put, get, size (remove не требуется, поддержка стандартного интерфейса Map не требуется).
Продемонстрируйте работоспособность реализации с помощью соответствующих тестов.
